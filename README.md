# nyhryan.github.io

This is a repository for my developer blog built with [Astro](https://astro.build/) webframework and GitHub Pages.

You can visit my blog here: [blog.ataidev.cc](https://blog.ataidev.cc/)

## Requirements

- [Bun](https://bun.com/)
- Powershell

## Development

```bash
# Install dependencies
$ bun install

# Run local preview server
$ bun run dev
# or if you don't don't have powershell
$ bunx astro dev

# Build to static files
$ bun run build
# or
$ bunx astro build
```

## Used libraries, packages

- [Astro](https://astro.build/) with TypeScript
- [tailwindcs](https://tailwindcss.com/)
- [Obsidian](https://obsidian.md/) for writing posts
- Visual Studio Code/JetBrains WebStorm
  - Profile for Visual Studio Code is included in [this repository](https://github.com/nyhryan/nyhryan.github.io/blob/main/Astro%20blog.code-profile).
